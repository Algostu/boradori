### 내풀이

`dp[i] = min(dp[0:i]) + 1` 이다. 다만 이것을 Bottom-up 방식으로 구현했다. 

```cpp
for(int i=1; i<=n; i++){
    int next[3] = { i+1, i*2, i*3 };
    for(int j=0; j<3; j++){
        if(next[j] <= n and (min_cost[next[j]] == 0 or min_cost[next[j]] > min_cost[i] + 1)){
            min_cost[next[j]] = min_cost[i] + 1;
            prev[next[j]] = i;
        }
    }
}
```
